## 🧪 開発者向け

このプラグインは PyQGIS + Qt Designer により開発されています。

### 必要な環境

- QGIS 3.x
- Python 3.7 以上
- PyQt5
- Qt Designer（UI 編集用）
- `pyrcc5`（リソースファイル変換用）

### リソースファイルのビルド

    ```bash
    pyrcc5 -o resources.py resources.qrc
    ```
